Требует Python 3.10.11.
Запуск осуществляется через ipynb файлы.

# Описание содержимого файлов и директорий

* Директория **Theory** включает в себя:
	* **Conditional_Distribution.pdf** - файл, в котором представлен вывод условной плотности для комплексного нормального распределения.
	* **Draft.pdf**- файл, в котором выведен ЕМ-алгоритм оценивания угловых координат источников сигнала и ковариации сигнала, модель сигнала стохастическая, матрица принятых сигналов сдержит пропуски. Шум предполагается известным.

* Директория **Program Complex for chapter I** включает в себя программный комплекс:
	* **common** -- включает в себя различные пакеты, которые используются в различных экспериментах, такие как:
		* **sensors.py.py** -- базовые функции для генерации сигналов и шумов, определения оценки ковариации по набору наблюдений с пропусками и т.д.
		* **debug_funcs.py** -- проверка размерности, отсутствия NaN, Inf, и в ряде случаев PD-свойства для матриц, возникающих на Е-шаге;
		* **hungarian_algorithm.py** -- решение проблемы label switching, при которой элементы вектора могут меняться местами между итерациями.
	* **Miscellaneous testing** -- реализует тестирование вспомогательных функций в базовых сценариях.
	* **Experimental Version** -- представлен алгоритм при сравнении с MUSIC, для улучшения эффективности инициализация основана на MUSIC + разброс. Правдоподобие монотонно возрастает!
