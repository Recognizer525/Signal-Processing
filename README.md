Требует Python 3.10.11.
Запуск осуществляется через ipynb файлы.

# Описание содержимого файлов и директорий

* Директория **Theory** включает в себя:
	* **Conditional_Distribution.pdf** - файл, в котором представлен вывод условной плотности для комплексного нормального распределения.
	* **Draft.pdf**- файл, в котором выведен ЕМ-алгоритм оценивания угловых координат источников сигнала и ковариации сигнала, модель сигнала стохастическая, матрица принятых сигналов сдержит пропуски. Шум предполагается известным.

* Директория **Program Complex for chapter I** включает в себя программные реализации ЕМ-алгоритма: 
**Actual Version**, где представлен алгоритм без сравнения с другими; 
**Experimental Version**, где представлен алгоритм при сравнении с MUSIC, для улучшения эффективности инициализация основана на MUSIC + разброс, кроме того, при приближении к оптимуму возможно накопление численных ошибок, поэтому если правдоподобие начинает падать - итерационный процесс прекращается. 
Общие для различных папок модули, такие как  **sensors.py**, **optim_doa.py**, **debug_funcs**, **angle_finding** и другие содержатся в директории **common**. Jupyter-файлы (ipynb) используются для тестирования ЕМ-алгоритма и вспомогательных функций.
